# Pre-Publish Checklist

Complete this checklist before publishing any content optimized for SEO, GEO, and conversion.

---

## Phase 1: Content Quality

### Accuracy & Value
- [ ] All facts verified from reliable sources
- [ ] Statistics include source citations
- [ ] Information is current (check dates)
- [ ] Content provides genuine value (not fluff)
- [ ] Unique insights or angle present
- [ ] Comprehensive coverage of topic (vs competitors)

### E-E-A-T Signals
- [ ] Author name and bio displayed
- [ ] Author credentials relevant to topic
- [ ] Experience signals present (case studies, examples)
- [ ] Expert quotes or references included
- [ ] Sources cited and linked
- [ ] "About" and "Contact" pages exist on site

### Readability
- [ ] Flesch-Kincaid grade appropriate for audience
- [ ] Paragraphs ≤ 4 sentences each
- [ ] Subheadings every 300 words maximum
- [ ] Bullet/numbered lists for scannable content
- [ ] Tables for comparative data
- [ ] Active voice used predominantly
- [ ] Jargon explained or avoided

---

## Phase 2: Traditional SEO

### Title Tag
- [ ] Contains primary keyword
- [ ] Keyword positioned early (front-loaded)
- [ ] 30-55 characters target (hard max 60 — count EVERY character including spaces and pipes)
- [ ] Compelling and click-worthy
- [ ] Brand name included ONCE (check for accidental duplication: "Brand | X | Brand")
- [ ] Character count verified after final edit

### Meta Description
- [ ] Contains primary keyword
- [ ] 120-155 characters (hard max 158 — count EVERY character)
- [ ] Includes value proposition
- [ ] Has call-to-action or hook
- [ ] Unique (not duplicated from other pages)
- [ ] Character count verified after final edit (re-count, don't trust initial count)

### URL Structure
- [ ] Contains primary keyword
- [ ] Short and readable (3-5 words max)
- [ ] Uses hyphens, not underscores
- [ ] No parameters or special characters
- [ ] Lowercase only

### Header Hierarchy
- [ ] One H1 per page (contains keyword)
- [ ] Minimum 4 H2s for landing pages/homepages, 3+ for articles over 500 words
- [ ] H3s for subsections under H2s
- [ ] Logical hierarchy (no skipped levels: H1→H3 without H2)
- [ ] Keywords distributed across headers

### Keyword Optimization
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword density 1-2% (natural)
- [ ] Secondary keywords distributed
- [ ] Semantic variations included
- [ ] No keyword stuffing

### Links
- [ ] 8+ internal links for landing pages/homepages, 3+ per 1000 words for articles
- [ ] At least 1 external link for landing pages, 2+ for articles (to authoritative sources)
- [ ] All links functional (no 404s)
- [ ] Link anchor text descriptive (not "click here")
- [ ] External links open in new tab

### Images
- [ ] All images have descriptive alt text
- [ ] Alt text includes keywords where natural
- [ ] Images compressed for fast loading
- [ ] File names descriptive (not IMG_001.jpg)
- [ ] Featured image set

---

## Phase 3: GEO Optimization

### Answer-First Structure
- [ ] Introduction contains direct answer to topic
- [ ] Each H2 section opens with key point
- [ ] Definitions provided on first mention
- [ ] Questions answered directly, then elaborated

### Self-Contained Chunks
- [ ] Each section understandable without context
- [ ] No ambiguous references ("as mentioned above")
- [ ] Key terms defined within section
- [ ] No assumptions about reader's prior reading

### Entity Clarity
- [ ] All entities named explicitly on first mention
- [ ] No ambiguous pronouns
- [ ] Relationships between entities stated
- [ ] Dates and numbers specific (not "recently")

### Citation Worthiness
- [ ] Quotable statements present
- [ ] Specific statistics with sources
- [ ] Original data or research (if applicable)
- [ ] Clear factual claims (not vague)

### FAQ Section
- [ ] FAQ section with 4-6 questions
- [ ] Questions match real user queries
- [ ] Answers are direct and complete
- [ ] FAQ schema markup implemented

### Schema Markup
- [ ] Article/BlogPosting schema added
- [ ] FAQ schema added (if FAQ section exists)
- [ ] HowTo schema (if step-by-step content)
- [ ] Schema validated with Google's tool

---

## Phase 4: Copywriting & Conversion

### Headline Effectiveness
- [ ] Creates curiosity or promises value
- [ ] Speaks to reader's needs/desires
- [ ] Uses power words appropriately
- [ ] Stands out from competitor headlines

### Introduction Hook
- [ ] First paragraph engages reader
- [ ] Problem or benefit stated early
- [ ] Reason to continue reading established
- [ ] Sets up what content delivers

### Value Communication
- [ ] Benefits emphasized (not just features)
- [ ] Reader addressed directly (you/your)
- [ ] Outcomes made tangible
- [ ] Objections anticipated and addressed

### Social Proof
- [ ] Testimonials included (if applicable)
- [ ] Case studies or examples present
- [ ] Statistics support claims
- [ ] Authority signals visible

### Call-to-Action
- [ ] Clear primary CTA present
- [ ] CTA uses action verb + benefit
- [ ] CTA visually prominent
- [ ] Friction minimized (explain what happens next)
- [ ] CTA appears above and below fold

---

## Phase 5: Technical Checks

### Page Speed
- [ ] Page loads in < 3 seconds
- [ ] Images optimized and lazy-loaded
- [ ] CSS/JS minimized
- [ ] Core Web Vitals passing (LCP, FID, CLS)

### Mobile Experience
- [ ] Fully responsive layout
- [ ] Text readable without zooming
- [ ] Buttons/links easily tappable (44px+)
- [ ] No horizontal scrolling
- [ ] Mobile preview checked

### Indexability
- [ ] No unintentional noindex tag or X-Robots-Tag header
- [ ] Canonical tag present with full absolute URL (`<link rel="canonical" href="https://..." />`)
- [ ] WWW canonicalization configured (www→non-www or vice versa, 301 redirect)
- [ ] Page accessible from internal links
- [ ] Page included in XML sitemap
- [ ] XML sitemap exists and is accessible at /sitemap.xml
- [ ] Sitemap submitted to Google Search Console AND Bing Webmaster Tools

### Bot Access
- [ ] robots.txt exists and is accessible at /robots.txt
- [ ] robots.txt references sitemap URL
- [ ] Bingbot allowed (critical for ChatGPT)
- [ ] OAI-SearchBot allowed
- [ ] Content not behind JavaScript-only rendering

### Open Graph & Social
- [ ] ALL required OG tags present: og:title, og:description, og:image, og:url, og:type, og:site_name
- [ ] Twitter Card tags present
- [ ] Social sharing preview verified

---

## Phase 6: Final Review

### Spelling & Grammar
- [ ] Spell-check completed
- [ ] Grammar review completed
- [ ] Consistency in terminology
- [ ] Proper capitalization

### Formatting
- [ ] Consistent heading styles
- [ ] Bullet/list formatting consistent
- [ ] Links formatted correctly
- [ ] No orphan or widow text issues

### Links Test
- [ ] All internal links work
- [ ] All external links work
- [ ] Links open in intended target

### Preview Check
- [ ] Preview on desktop
- [ ] Preview on mobile
- [ ] Social sharing preview correct
- [ ] SERP preview acceptable
