# Article Template (Dual-Optimized)

Use this template for blog posts and articles optimized for both traditional SEO and GEO.

---

## Pre-Writing Checklist

- [ ] Primary keyword identified
- [ ] Search intent analyzed (SERP analysis complete)
- [ ] Secondary keywords mapped
- [ ] Target word count determined (based on competitors)
- [ ] Unique angle/value proposition defined
- [ ] Target audience persona referenced

---

## Post-Writing Checklist

### SEO Checklist
- [ ] Primary keyword in title (front-loaded)
- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Secondary keywords distributed naturally
- [ ] Meta description includes keyword + CTA
- [ ] URL is short and includes keyword
- [ ] Images have descriptive alt text
- [ ] Internal links to related content (3-5)
- [ ] External links to authoritative sources (2-3)

### GEO Checklist
- [ ] Answer-first format in all sections
- [ ] Each H2 section is self-contained
- [ ] Entity definitions are explicit
- [ ] Statistics have sources cited
- [ ] FAQ section with schema markup
- [ ] Article schema implemented
- [ ] No ambiguous pronouns or references

### Copywriting Checklist
- [ ] Headline creates curiosity or promises value
- [ ] Introduction hooks the reader
- [ ] Benefits clearly articulated
- [ ] Social proof/authority signals included
- [ ] Clear CTA at the end
- [ ] Readable: short paragraphs, subheadings

### Readability Checklist
- [ ] Paragraphs ≤ 4 sentences
- [ ] Subheadings every 300-400 words
- [ ] Bullet points for lists
- [ ] Tables for comparative information
- [ ] Active voice dominant
- [ ] Jargon explained or avoided
