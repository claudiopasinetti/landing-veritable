## Adding References

To add a reference:

1. Take/find a screenshot of the element
2. Name it: `[element]-[variant]-[number].png`
3. Place in `references/` folder
4. Update this file with the path

Example:
```markdown
| Minimal | Logo + few links + CTA | [nav-desktop-minimal-01.png](../references/nav-desktop-minimal-01.png) |
```
